HTML
HTML (Hypertext Markup Language) é a linguagem de marcação usada para estruturar e organizar o conteúdo de uma página web. Ele define os elementos como títulos, parágrafos, imagens e links, que formam a base da página.
CSS
CSS (Cascading Style Sheets) é uma linguagem usada para controlar o design e a aparência das páginas web. Com o CSS, é possível definir cores, fontes, espaçamento e layout, permitindo personalizar a estética da página e torná-la mais agradável aos olhos dos usuários.
Java script
JavaScript é uma linguagem de programação que permite adicionar interatividade e dinamismo às páginas web. Com o JavaScript, é possível criar funcionalidades como animações, validações de formulários e atualizações de conteúdo em tempo real, melhorando a experiência do usuário.
